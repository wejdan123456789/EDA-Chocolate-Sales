#  Chocolate Sales Data Analysis (EDA)

## Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on a chocolate sales dataset.  
The goal is to understand the structure of the data, clean it, and extract meaningful insights that can support business decisions.

---

##  Dataset Description
The dataset contains information about chocolate sales transactions, which may include:

- Product names
- Sales amounts
- Regions or locations
- Dates of sales
- Additional attributes related to customer or transaction details

This dataset is used to analyze sales patterns and identify trends.

---

##  Objectives
The main objectives of this analysis are:

- Understand the dataset structure and variables
- Handle missing or inconsistent data
- Explore distributions of key variables
- Analyze relationships between features
- Identify trends and patterns in sales

---

## Tools & Technologies
The following tools and libraries were used:

- Python
- Pandas (data manipulation)
- Matplotlib (visualization)
- Seaborn (advanced visualization)
- Jupyter Notebook

---

## Data Cleaning
During preprocessing, the following steps were performed:

- Handling missing values
- Removing duplicates (if any)
- Converting data types where necessary
- Formatting columns for consistency

---

## 📈 Exploratory Data Analysis (EDA)

The analysis includes:

- Distribution of sales values
- Most sold products
- Sales comparison across regions
- Visualization of trends using charts and graphs

---

## Key Insights
Some important findings from the analysis include:

- Certain products have significantly higher sales than others
- Sales distribution shows variation across regions
- Patterns in the data may indicate seasonal or demand-based trends
